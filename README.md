# Clicker Rewrite

A rewrite of one of my first projects.

## Dependencies

- [Rokit](https://github.com/rojo-rbx/rokit)
- [npm](https://www.npmjs.com/package/npm)
- [git](https://github.com/git-guides/install-git)

### Windows/MacOS

- [Roblox Studio](https://create.roblox.com/docs/en-us/tutorials/curriculums/studio/install-studio)

### Linux

- [Vinegar](https://vinegarhq.org/Vinegar/Installation/index.html)

## Usage

First, clone the repository and install the dependencies.

```console
git clone https://github.com/swordofastrea/clicker-rewrite.git
npm i
```
